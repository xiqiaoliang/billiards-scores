# 台球追分计分规则说明（2人 + 3人）

> 本文档基于当前讨论结论整理，用于统一产品规则口径（不依赖当前代码实现）。

## 1. 总体规则

- 支持两种模式：`二人追分`、`三人追分`。
- 默认新比赛模式：`二人追分`。
- 点击「新比赛」时，需选择 `二人追分` 或 `三人追分`。
- 三人模式默认选手名：`吴`、`席`、`王`。
- 所有模式下，每局必须且只能有一个胜负项（含普通/让杆/黑金形态）。

## 2. 分值定义

- 犯规：1
- 让杆犯规：1
- 开球犯规：1
- 分球：2
- 普胜：4
- 黄金9：4
- 小金：7
- 大金：10

## 3. 二人追分规则

### 3.1 基础结算

- 记分关系为双方对冲（零和）。
- 任意标签结算后，本局两人净分之和必须为 0。

### 3.2 让杆/黑金

- 允许让杆与黑金同时勾选。
- 让杆状态下，按“下家”结算；二人模式下下家即对手。
- 让杆得分区可选：`犯规`、`分球`、`普胜`、`小金`、`大金`。
- 勾选让杆时隐藏 `黄金9`。
- 若让杆与黑金同时勾选，则隐藏 `犯规`、`分球`，只可选：`普胜`、`小金`、`大金`。

### 3.3 交互约束

- 开球犯规、分球、胜负项均支持快速切换（点 A 后再点 B，应直接替换为 B）。

## 4. 三人追分规则

## 4.1 顺序定义

- 记当前击球顺序为循环队列，例如 `A -> B -> C -> A`。
- 对当前选手 X：
  - 上家：X 的前一位
  - 下家：X 的后一位

## 4.2 基础结算方向

- 犯规、开球犯规：赔上家。
- 分球、普胜：赢上家。
- 黄金9、小金、大金：向上下两家结算（上下家都参与）。

## 4.3 让杆规则

- 让杆含义：下家不打，本家继续打。
- 三人模式增加 `让杆犯规`（仅让杆状态可选，分值 1，赔下家）。
- 让杆得分区可选：`犯规`、`分球`、`普胜`、`小金`、`大金`。
- 勾选让杆时隐藏 `黄金9`（黄金9/黑黄金9都视为开球进9，不存在让杆）。
- 让杆与黑金可同时勾选；同时勾选后隐藏 `犯规`、`分球`，仅保留：`普胜`、`小金`、`大金`。
- 让杆状态下，本家正向得分翻倍（计入额外得分）；若本家为赔分方，则赔分仍按原分值结算。

## 4.4 黑金规则

- 黑金普胜：只赔上家。
- 黑金黄金9 / 黑金小金 / 黑金大金：赔上下两家。
- 若同时满足“让杆 + 黑金”，按让杆优先：只与下家结算。

## 4.5 快速切换

- 三人模式下，开球犯规 / 分球 / 胜负项都支持快速切换。
- 例如先点 `A 普胜`，再点 `B 普胜`，应直接替换为 `B 普胜`，不弹“只能存在一个”的阻断提示。

## 4.6 局后顺序与开球

- 常规胜局：获胜者与上家交换顺序，下一局由获胜者开球。
- 黑金或让杆获胜：不交换击球顺序。
- 黑金局下一局由黑金者开球；让杆局按获胜者继续开球。

## 5. 提交与展示规则

## 5.1 提交区标签显示

- 让杆犯规标签显示为：`让杆犯规`。

## 5.2 逐局记录（小计区）

- 每位选手逐局需显示：
  - 所有得分项（只展示造成该选手得分的项）
  - 小计
  - 净分
- 小计定义：该选手本局所有“得分项”的和，`小计 >= 0`。
- 净分定义：该选手本局最终净得分，可为负数。
- 若无得分项，则小计显示 `0`。
- 得分项文案只显示“标签名/来源选手+标签名”。

## 5.3 三人记录顺序展示

- 记分操作区：按当前局击球顺序展示选手栏位。
- 总览区：固定按选手身份顺序展示，不随击球顺序变化。
- 逐局历史：每位选手小计按当局击球顺序依次展示，每人占整行。

## 6. 总览统计口径

- 每位选手初始总分：`100`。
- 每局结束后：总分 = 100 + 累计净分。
- 总览净分列：显示各选手当前累计净分（初始为 0）。
- 守恒约束：
  - 每局所有选手净分和 = 0
  - 总览所有选手净分合计 = 0
  - 总览所有选手总分合计 = 100 × 人数

## 7. 二维码规则

- 当存在未推送远程的代码时版本信息保持不变。
- 不因本轮规则调整升级版本号。
